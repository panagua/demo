Style guide

  1. use lowercase element name
  2. close all html tag/elements
  3. use lowercase attribute names
  4. always quote attribute value
  5. always specify alt, width, and height for images
  6. spaces and equal signs
  7. avoid long code line
  8. never skip the <title> element because required for SEO! so try to make title as accurate and meaningful as possible
  9. lang to set languange
  10. charset
  11. viewport
  12. html comments
  13. use lowercase nfile name
  14. extension:
    1. html as .html or htm
    2. css files as .css
    3. javascript as .js
